Bot Commands

Command: /add_update

Description: Adds an update for a specific version. Usage: `/add_update version update_details`.

Command: /add_bug

Description: Adds a bug with a specific ID. Usage: `/add_bug bug_id bug_details`.

Command: /remove_entry

Description: Removes an update or bug. Usage: `/remove_entry entry_type entry_id`.

Command: /show_all

Description: Shows all updates and bugs.

Command: /setup_civilian

Description: Sets up the 'Civilian' role.

Command: /setup_leo

Description: Sets up the 'Law Enforcement Officer' role.

Command: /setup_ems

Description: Sets up the 'Emergency Medical Service' role.

Command: /setup_fd

Description: Sets up the 'Fire Department' role.

Command: /check_vitals

Description: Checks the vitals of a specified player

Command: /do

Send an interaction about what you are doing

Command: /me

Send an interaction about what you are feeling

Command: /toggle_bodycam

Toggle your bodycam status

Command: /toggle_dashcam

Toggle your dashcam status

Command: /cuff

Cuff a player

Command: /ziptie

Ziptie a player

Command: /uncuff

Uncuff a player

Command: /cut_ziptie

Cut off a ziptie from a player

Command: /search_vehicle

Search a player's vehicle

Command: /search_player

Search a player
